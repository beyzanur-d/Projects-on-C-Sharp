# WPF Canvas – Araba ve Çiçek Animasyonu

C# ve WPF kullanılarak geliştirilmiş basit bir Canvas uygulaması.
Araba ve çiçek şekilleri çizilir; klavye ile hareket ettirilebilir.

![Tasarım](https://github.com/beyzanur-d/Projects-on-C-Sharp/image.png)

## Özellikler

- Canvas üzerinde araba (gövde, cam, tekerlekler) ve çiçek (merkez + 8 yaprak) çizimi
- WASD tuşlarıyla tüm şekilleri birlikte hareket ettirme
- Canvas sınırlarını aşmayı engelleyen kontrol mekanizması

## Kontroller

| Tuş | Hareket |
|-----|---------|
| W   | Yukarı  |
| A   | Sol     |
| S   | Aşağı   |
| D   | Sağ     |

## Gereksinimler

- .NET Framework 4.7.2 veya üzeri
- Visual Studio 2019 / 2022

## Çalıştırma

1. Projeyi klonlayın veya ZIP olarak indirin
2. `Test3.sln` dosyasını Visual Studio ile açın
3. `F5` ile derleyip çalıştırın
